# SyncSound

`SyncSound` - настольное приложение на `C++` и `Qt Widgets` для локального воспроизведения музыки с плейлистом.

На текущем этапе проект представляет собой графический аудиоплеер с заготовками интерфейса под будущую сетевую синхронизацию. Основной упор сейчас сделан на локальную работу с треками и базовые элементы управления воспроизведением.

## Технологии

- `C++17`
- `Qt Widgets`
- `Qt Multimedia`
- `CMake`

## Требования

Для сборки нужны:

- `CMake` версии `3.16` или новее;
- `Qt 5` или `Qt 6` с модулями `Widgets` и `Multimedia`;
- компилятор с поддержкой `C++17`.

## Сборка

Пример стандартной сборки через `CMake`:

```bash
cmake -S . -B build
cmake --build build
```

После сборки исполняемый файл будет создан в каталоге `build/`.

## Запуск

После старта приложения можно:

1. Нажать `Add` и выбрать локальные аудиофайлы.
2. Выбрать трек в списке или открыть его двойным кликом.
3. Управлять воспроизведением через кнопки и ползунки в нижней части окна.

Поддержка форматов зависит от установленной версии `Qt Multimedia` и доступных мультимедийных backend'ов в системе.

## Текущее состояние

Сейчас `SyncSound` - это рабочий прототип локального аудиоплеера с UI для будущего расширения. Блоки, связанные с совместным прослушиванием, пока являются заготовками интерфейса и не подключены к сетевой логике.
