# sync-music-player — Синхронный музыкальный плеер

В программе выбирается роль (админ/хост или слушатель). Хост выступает в роли сервера и управляет воспроизведением. Слушатели подключаются к нему по какому-то идентификатору, например, по IP. Музыкальные файлы для простоты можно брать из подпапки проекта вроде music. Будем считать, что пользователи находятся в одной локальной сети

> [!WARNING]
> Текущая версия кода является ранним прототипом (скелетом приложения).<br>
> Основные функции на данном этапе ещё не реализованы и находятся в процессе активной разработки.<br>
> Этот коммит содержит структуру проекта, которая может и будет изменена в процессе дальнейшей разработки.

## Стек технологий и окружение

### Основной стек
* Язык: C++20
* Система сборки: CMake 4.3.4

### Окружение и инструменты
* Компилятор: GCC / G++ (в составе подсистемы MSYS2 MinGW-w64)
* Среда разработки (IDE): VS Code (с расширениями C/C++ и CMake Tools)
